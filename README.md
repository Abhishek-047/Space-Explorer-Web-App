# 🌌 Space Explorer Web App

## 🚀 Overview
Space Explorer is a premium web-based application that allows users to explore space imagery using **NASA APIs**.  
It provides a **cinematic UI experience** with real-time search, dynamic rendering, and an interactive landing page.

---

## 🎯 Project Objective
This project demonstrates:

- JavaScript fundamentals (DOM manipulation, events)
- API integration using Fetch API
- Dynamic UI rendering
- Use of higher-order functions (`filter`, `sort`, `slice`)
- Clean UI/UX design with modern frontend practices

---

## 🔌 APIs Used

### 1. NASA Open API (API Key Required)
https://api.nasa.gov/  

- Used via personal API key  
- Provides general NASA data (like APOD, etc.)

---

### 2. NASA Image and Video Library API (No API Key Required)
https://images-api.nasa.gov/  

- Used for searching and fetching space images dynamically  
- Main API used for rendering search results  

---

## ⚙️ API Handling Features

- Fetch API for data requests  
- AbortController to cancel stale requests  
- Error handling for failed API calls  
- Optimized search using filtering, sorting, and slicing  

---

## ✨ Features

### 🚀 Core Features
- 🔍 Search any space object (planet, galaxy, nebula, etc.)
- 🖼️ Dynamic image results from NASA API
- ⭐ Featured result (highlighted main image with details)
- 📊 Filtering + Sorting + Slicing (using HOFs)
- ⌨️ Search via button + Enter key
- ❌ Error handling + loading states
- ⚡ Debouncing (optimized performance)
- 🧠 AbortController (cancels previous API calls)

---

### 🎨 UI / UX Features
- 🌌 Cinematic landing page (hero section)
- 🪐 Glowing planet visual with animation
- 💫 Smooth transitions & hover effects
- 📱 Fully responsive design
- 🎯 Split layout (left: info, right: visuals)
